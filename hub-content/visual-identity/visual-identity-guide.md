# Guide: Visual Identity

This guide explains how to use your brand assets correctly.

Your visual identity includes:

- Logo files
- Color palette
- Typography
- Templates
- Brand guidelines

Using these correctly protects your brand and keeps everything consistent.

---

## What this is

Your visual identity defines how your company looks.

It helps you:

- Build trust
- Look professional
- Stay consistent
- Be recognizable

You received your brand assets as part of your project delivery.

---

# Part 1 — Using Your Logo Correctly

## Logo File Types Explained

You received multiple versions of your logo.

### SVG (Vector format)

Best for:

- Website
- Large prints
- Professional printing
- Scaling to any size

Never loses quality.

---

### PNG

Best for:

- Social media
- Presentations
- Documents
- Transparent backgrounds

Good for digital use.

---

### JPG

Best for:

- Simple digital sharing
- Email attachments

Not ideal for print.

---

## Which Logo Version Should I Use?

You may have received:

- Primary logo
- Horizontal version
- Stacked version
- Icon-only version
- Light version (for dark backgrounds)
- Dark version (for light backgrounds)

### Use the primary logo by default.

Use variations only when:

- Space is limited
- Background requires contrast
- Social media profile picture needs icon-only version

---

## Do Not

- Stretch the logo
- Change its colors
- Add shadows or effects
- Place it on low-contrast backgrounds
- Crop parts of the logo

If unsure, ask.

---

# Part 2 — Colors & Typography

## Color Palette

Your brand includes defined colors.

Use:

- Primary color for main highlights
- Secondary colors for support
- Neutral colors for background

Avoid:

- Random new colors
- Overuse of strong accent colors

If you are unsure, follow your brand guide PDF.

---

## Typography

If fonts were delivered:

- Use the defined primary font for headings
- Use the secondary font for body text

If the brand includes system-safe alternatives,
use those in email or internal documents.

Do not mix random fonts.

---

# Part 3 — Templates & Print Material

## Business Cards

Delivered as:

- Print-ready PDF
- Digital formats

For printing:

- Send the print-ready PDF to the printer
- Do not modify bleed or margins

---

## Letterhead

Use:

- Official template
- Correct logo placement
- Correct spacing

Avoid resizing elements.

---

## Presentation Templates

Delivered as:

- PPTX file
- Possibly PDF version

Use:

- Master slides
- Predefined layouts
- Correct font styles

Do not:

- Insert different fonts
- Change layout structure

---

## Social Media Assets

Includes:

- Profile image templates
- Post templates
- Highlight covers (if included)

Use these to maintain visual consistency.

---

# Part 4 — Brand Consistency

Consistency builds recognition.

Inconsistent branding:

- Looks unprofessional
- Confuses customers
- Weakens trust

Ask yourself:

- Does this follow our brand guide?
- Does it look like our website?
- Would someone recognize it instantly?

If not, adjust.

---

# File Management Best Practice

Store your brand assets:

- In one shared company folder
- With original filenames
- Without renaming the master files

Create copies when editing.

Never edit the original master files.

# Technical Information (Advanced)

### File Formats

SVG:

- Vector-based
- Infinite scalability
- Ideal for professional print

PNG:

- Raster format
- Supports transparency
- Suitable for digital platforms

JPG:

- Compressed raster format
- Smaller file size
- Not ideal for professional print

### Color Systems

You may have received:

- HEX codes (web)
- RGB values (digital)
- CMYK values (print)

Always use CMYK for print production.

### Print Considerations

- Use high-resolution files
- Do not rescale manually
- Ensure correct bleed if printing

Professional printers prefer:

- Vector files (SVG/PDF)
- CMYK color profile
